# 📊 Flipkart Customer Support Analysis & CSAT Prediction

## 📌 Project Overview

Customer support is a critical component of customer satisfaction and business success. This project analyzes Flipkart Customer Support data to understand customer interactions, support performance, response efficiency, and satisfaction levels. Through Exploratory Data Analysis (EDA) and Machine Learning (ML), the project uncovers actionable insights and explores predictive modeling for customer satisfaction.

---

## 🎯 Objectives

* Analyze customer support operations and service quality.
* Identify factors affecting Customer Satisfaction (CSAT).
* Evaluate support channel performance.
* Examine agent and shift effectiveness.
* Analyze response time impact on customer satisfaction.
* Build a Machine Learning model to predict CSAT scores.

---

## 📂 Dataset Features

The dataset contains information related to:

* Customer support channels
* Product categories
* Customer satisfaction scores (CSAT)
* Agent performance
* Agent shifts
* Agent tenure
* Issue reporting and resolution timestamps
* Response times

## 🔍 Key Insights

* Customer satisfaction varies across support channels.
* Response time significantly impacts CSAT scores.
* Certain product categories generate more support requests.
* Agent experience influences customer satisfaction.
* Some shifts consistently achieve better CSAT performance.

## 💡 Business Recommendations

* Reduce customer response times.
* Improve training for underperforming agents.
* Optimize staffing across support channels.
